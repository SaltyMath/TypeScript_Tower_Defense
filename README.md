# TypeScript_Tower_Defense

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Build the project:

```bash
npm run build
```

3. Start the local dev server based on dist folder:

```bash
npm run serve
```

4. Open the game in your browser:

- `http://127.0.0.1:8080`

## Development

- Watch TypeScript changes and rebuild automatically:

```bash
npm run watch
```

- If you want to keep the server and watch process open, run the build command first and then serve.

## Notes

- The game entry point is `index.html`.
- Compiled files are written to the `dist/` folder.
